# homebrew-tap

Homebrew tap for tools by [@cpeoples](https://github.com/cpeoples).

## Install

```sh
brew install cpeoples/tap/ansible-security-scanner
```

[`ansible-security-scanner`](Formula/ansible-security-scanner.rb) is a static
security scanner for Ansible playbooks, roles, and collections. See the
[upstream repository](https://github.com/cpeoples/ansible-security-scanner)
for usage and rule documentation.
